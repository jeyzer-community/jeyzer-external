
------------------------------------------------------------------------------------------------
                            External profile repository
------------------------------------------------------------------------------------------------

This profile repository is a place holder for your master and shared profiles.
Best practice is to store it in your own SCM repository and check it out here.
These profiles must be compatible with the Jeyzer project version.

To build a new master profile, copy paste an existing one or use a master template available in the master/template directory.
See https://jeyzer.org/docs/master-profile/creation-guidelines/

To build a new shared profile, copy paste an existing one or use a shared template available in the shared/template directory.
See https://jeyzer.org/docs/shared-profile/creation-guidelines/
